# Immutablue Docs

Welcome to Immutablue, an easy, extendable version of Fedora Silverblue with sane defaults built for the future with cloud-native practices.

Immutablue is designed to provide a stable, immutable base OS with flexible application layers, allowing you to enjoy the benefits of immutability without sacrificing customization or usability.

## Relevant Links
- [GitLab](https://gitlab.com/immutablue/immutablue)
- [Create an issue](https://gitlab.com/immutablue/immutablue/-/issues/new)

## Key Features

- **Immutable base system** with transactional updates
- **Multiple desktop environments** including GNOME (Silverblue), KDE (Kinoite), Sway, and more
- **Specialized variants** for hardware platforms like Apple Silicon (M1/M2/M3) and NVIDIA GPUs
- **Layered package management** using rpm-ostree, Flatpak, Distrobox, and Homebrew
- **Automated setup** and maintenance with systemd services and timers
- **Highly customizable** via configuration files and file overrides
- **Cloud-native practices** leveraging container technologies
