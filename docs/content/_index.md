## Key Features

- **Immutable base system** with transactional updates
- **Multiple desktop environments** including GNOME (Silverblue), KDE (Kinoite), and more
- **Specialized variants** for NVIDIA GPUs, Apple Silicon, Kubernetes, and ZFS
- **Layered package management** using Flatpak, Distrobox, Homebrew, and rpm-ostree
- **Automated setup** and maintenance with systemd services and timers
- **Highly customizable** via configuration files and build system
