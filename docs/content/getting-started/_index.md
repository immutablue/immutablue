## What is Immutablue?

Immutablue is an easy, extendable version of Fedora Silverblue with sane defaults built for the future with cloud-native practices. It provides:

- **Immutable base system** - The core OS is read-only and updated atomically
- **Layered package management** - Install software via Flatpak, Distrobox, Homebrew, or rpm-ostree
- **Automated setup** - First-boot wizard configures your system
- **Easy updates** - Single command updates everything
